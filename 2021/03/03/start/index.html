文字记录，不立 Flag

2021-03-03 | 生活随想 | 434 字 | 1 分钟

书读百遍其义自现
好记性不如烂笔头

长久以来都想着提笔来记录平日的工作和生活，总是动力不足。断断续续尝试了不同的文字输出工具，经历一番折腾，最终选择使用 Hexo + Git 来搭建一个记录工具。

以往总是觉得互联网的信息随手可得，中意的内容收藏起来分分钟。但是缺少整理和思考，这些好的内容便很快消失殆尽。思考的结果如果没有留下印记，也很快会被替代。这个时候便觉得文字本身是个好东西。

文字是好，但是使用什么样的方式记录。手写？难以定位和检索。文档？整理耗时。反倒 Blog 这种“互联网”早期产物，愈久弥新。就如目前我选用的这个记录方式，借用 Git 每篇文章都可以不停的更新迭代，除了自然的时间维度，每篇文字都有它的生命周期。

某种意义上可以认为文字活了。活的文字，也许才能带来某种交流的快感，这是自我的交流，且以为始终提升的手段。

文字的内容可以是多种多样的，但是为了聚焦，还是集中在各种 Hack 的思考。工作的、生活的，问题总是接踵而至，希望花式 Hack 能够带来不一样的人生体验。

本来想立个 Flag，比如一天一篇啥的，想想还是算了，喜欢的事情自然而然就做了，反倒少了逼迫感。就权当一个检验工具吧，希望自己是真的喜欢文字。
